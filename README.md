# ClimaxCombo.com

## Project Description
This is a webpag that compiles information from multiple high level tournamnts for the Weiss Schwarz trading card game and displays data based on tournament results and visualises the current metagame.

## Project Tasks
- **Week 1: Set up the development environment**
  - set up webpges in IDE
  - Configure Git and GitHub repository
- **Task 2: Plan the project**
  - Create the webpages used in the project and set up some sample data
- **Task 3: Gather information**
  - Create database and input tournament data
- **Task 4: Input data**
  - Input the tournament data from databases into webpage and create graphs to display data
- **Task 5: Host Webpage**
  - move files to web hosting site and launch webpage 

## Project Skills Learned
- Git and GitHub for version control and collaboration
- Project planning and task management
- Software development
- Writing documentation
- Database management
- Web hosting skills
